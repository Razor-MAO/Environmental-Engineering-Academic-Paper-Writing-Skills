## Lab Paper Polisher 🧪📄

A Codex skill for polishing English academic paragraphs — sentence-structure diversity, research-group style alignment, and structured revision notes.

### What it does

- Rewrites academic paragraphs into publication-ready language without changing facts, data, or conclusions
- Replaces AI-sounding phrasings with natural academic English
- Aligns with a target research group's writing style using customizable style profiles
- Outputs a revision table explaining every change and its basis

### How to use

1. Paste one or more English academic paragraphs.
2. Optionally specify which lab or journal style to follow.
3. Get a polished version + a structured revision log.

### Background

Selecting outstanding papers from my research group in recent years, I distilled the skills summarized by AI tools. We will continuously iterate based on actual user experience.
